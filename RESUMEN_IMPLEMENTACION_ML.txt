📋 PASOS QUE DEBES COMPLETAR
═════════════════════════════════════════════════════════════════════════════

⚠️ IMPORTANTE: Aún faltan estos pasos para que funcione:

1️⃣ EJECUTAR SCRIPT SQL
   📁 Archivo: SQL/00_MercadoLibre_Tabla_y_SPs.sql
   🎯 Acción: Ejecutar en SQL Server Management Studio
   ✅ Resultado: Crea tabla y stored procedures

2️⃣ CREAR APLICACIÓN EN MERCADO LIBRE
   🌐 URL: https://developers.mercadolibre.cl/devcenter/
   🎯 Acción: Crear nueva aplicación
   ✅ Resultado: Obtener CLIENT_ID y CLIENT_SECRET

3️⃣ CONFIGURAR CREDENCIALES
   📁 Archivo: appsettings.json
   🎯 Acción: Reemplazar CLIENT_ID y CLIENT_SECRET
   ✅ Resultado: App configurada

4️⃣ AGREGAR CAMPO EN PROPIEDADES (OPCIONAL)
   📁 Tabla: Propiedades
   🎯 Acción: Agregar columna ItemIdMercadoLibre
   ✅ Resultado: Poder guardar IDs de Mercado Libre

5️⃣ AGREGAR BOTONES EN VISTAS (OPCIONAL)
   📁 Archivo: Views/Propiedades/Index.cshtml
   🎯 Acción: Agregar botón para ver visitas
   ✅ Resultado: Ver visitas desde módulo de propiedades
